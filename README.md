# Applicable Scope of Multiplicative Entropy

A system or a physical model can adopt a multiplicative entropy description if it satisfies the following conditions:

1.The number of units carrying numerical values remains constant.

2.Each unit is associated with a quantifiable value (e.g., energy, resource amount, or population size).

3.The total sum of all unit values is conserved.

4.The dynamical evolution follows a flow or exchange rule wherein quantities transfer from units with higher values to those with lower values.

5. The numerical value carried by each unit must be unitized, i.e., expressed as an integer multiple greater than 1 of the basic unit.In other words, the value carried by any unit is a positive integer.
